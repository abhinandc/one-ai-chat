# OneEdge - OneOrigin's AI Hub

OneEdge is an employee-facing AI platform that provides enterprise teams with controlled access to multiple AI models.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```

## Environment Variables

Create a `.env` file with the following variables:

```
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_API_PROXY_URL=
VITE_GOOGLE_CLIENT_ID=
```
